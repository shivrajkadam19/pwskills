3D MODELING Competition (Catia v5)

About Event : This event is organized to test designing skills and knowledge of creation of 3 D model from 2D drawing of machine components by participants using ‘CATIA’ designing software application in a live competition.

Rules & Regulations :

  • Participant need to create 3-D model from drawing given by judge in 30 min.
  • Evaluation of the model will be done on the basis of 3-D model accuracy and Completion of 3D Model.

Fees : Rs. 100
